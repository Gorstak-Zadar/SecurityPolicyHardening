# 🔐 SecurityPolicyHardening

> **Security Policy Hardening** - Applies comprehensive security policy hardening.

---

## 📋 Overview

SecurityPolicyHardening applies comprehensive Windows security policy hardening including password policies, audit policies, and user rights assignments.

---

## 🎯 What It Does

- 🔐 **Policy Hardening** - Secure configurations
- 🔑 **Password Policy** - Strong requirements
- 📝 **Audit Policy** - Event logging
- 👤 **User Rights** - Access controls

---

## 🚀 Usage

```powershell
# Apply policy hardening
.\SecurityPolicyHardening.ps1
```
